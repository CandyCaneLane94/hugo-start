Prodej alb Nirvany

(https://www.statisticbrain.com/nirvana-album-sales-statistics/)

Tento web poskytuje data o prodeji studiových, live a kompilačních alb kapely Nirvana a dále také zahrnuje data o prodeji box setů a jednotlivých singlů. U jednotlivých alb, singlů a box setů jsou informace o počtu prodaných kusů, roku vydání a umístění v žebříčku. V další části jsou mimo jiné i přesné počty vydaných alb, singlů, hudebních videí atd. Data by mohla být využita pro lepší orientaci čtenáře v oblasti tvorby kapely a pro přehled popularity jednotlivých alb a singlů. Jelikož jsou tyto data fakta, mohou být publikována kýmkoliv.
